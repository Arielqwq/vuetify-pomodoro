## **使用說明**
<ol>
<li>每次行程為倒數計時 25 分鐘，時間到後，將有 5 分鐘的休息時間 (休息時間可跳過)。</li>
<li>當行程尚未開始前，可編輯行程名稱。</li>
<li>預設鈴聲有 2 種，可任意選擇更換。</li>
</ol>
